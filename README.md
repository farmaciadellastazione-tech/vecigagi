# LinguaeQuiz 🌍

**Multilingual vocabulary quiz with spaced repetition**

A free, open-source web app to learn and compare vocabulary across multiple languages simultaneously — including classical languages and regional dialects.

👉 **[Try it live](https://farmaciadellastazione-tech.github.io/linguaequiz)**

---

## Features

- **890+ vocabulary entries** organized by theme and CEFR level (A1→B2)
- **16 languages** — Italian, English, French, Spanish, Portuguese, German, Korean, Chinese, Japanese, Arabic, Latin, Ancient Greek, Interlingua, Genoese, Spezzino, Carrarese
- **6 study modes** — Quiz, Training, Listen & Write, Voice, Dictation, Non-stop
- **Spaced repetition** — cards reappear at increasing intervals (1, 3, 7 days)
- **Verb conjugation tables** — essere, avere, amare in 9 languages across 6 tenses
- **Non-stop mode** — answer correctly as many times as possible before making a mistake, with personal record tracking
- **Dark theme** — easy on the eyes
- **No installation** — single HTML file, runs in any modern browser
- **No account required** — progress saved locally
- **Export/import** — save and restore your progress as JSON

## How to use

1. Open the app in your browser
2. Select your active languages (✎ button)
3. Choose a study mode
4. Select source and target language (or leave on random)
5. Study!

Progress is saved automatically in your browser's localStorage. Use 💾 to export and 📂 to import your progress.

---

## Technical notes

- Single HTML file (~280KB)
- React 18 (CDN) + Babel + Tailwind CSS
- No backend, no server, no cookies
- Web Speech API for pronunciation and voice recognition (Chrome recommended)

## Contributing

Translation errors are likely, especially in less common languages. If you spot a mistake, please open an **Issue** on GitHub with:
- The word/phrase
- The language
- The correct translation

Vocabulary suggestions and dialect contributions are very welcome.
